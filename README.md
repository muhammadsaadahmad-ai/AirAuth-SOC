# AirAuth-SOC

AirAuth-SOC is a touchless gesture-based authentication and security interaction prototype built with Python, OpenCV, and MediaPipe.

## Features
- Real-time hand tracking using webcam
- Air drawing with index finger
- Gesture-based mode switching
- Security-themed lock/unlock interaction
- Logging of gesture attempts

## Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy
